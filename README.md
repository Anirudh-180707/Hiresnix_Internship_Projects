# Full Stack Web Development Projects

This repository contains three full-stack web development projects completed as part of my internship. Each project demonstrates practical implementation of frontend development, backend APIs, database management, authentication, and responsive user interface design using modern web technologies.

## Projects

### 📦 Project 1 – Primary Inventory Management System
A web application for managing products, inventory, purchases, sales, and stock reports.

### 🏠 Project 2 – Real Estate Property Listing Portal
A property listing platform with advanced search, location-based features, and agent management.

### 🛒 Project 3 – Multi-Vendor E-Commerce Marketplace
An e-commerce platform supporting multiple vendors, product management, and secure online transactions.
