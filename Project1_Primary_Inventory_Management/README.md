# 📦 Primary Inventory Management System

A modern Inventory Management System frontend developed using **React.js** and **Vite**. The application provides an intuitive interface for managing products, monitoring stock levels, viewing purchase and sales records, and displaying inventory statistics through a clean dashboard.

---

## 🚀 Features

- Dashboard with inventory summary cards
- Product inventory management
- Product search functionality
- Add Product form
- Stock status indicators
- Purchase orders page
- Sales orders page
- Inventory reports dashboard
- Responsive sidebar navigation
- Modern and user-friendly interface

---

## 🛠️ Tech Stack

- React.js
- Vite
- JavaScript (ES6+)
- HTML5
- CSS3

## ⚙️ Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project directory:

```bash
cd Project1_Primary_Inventory_Management
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```
